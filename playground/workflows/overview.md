# Playground Overview

This directory (`./playground/`) is your persistent working space. Files written here survive across sessions and can be read by future sessions of this or other chats.

## Structure

```text
playground/
  workflows/      ← instructions for how to operate (you are here)
  research/       ← research notes, organized by date (YYYY-MM-DD/)
  projects/       ← long-running project context and notes
```

Subdirectories are created on demand — if a directory doesn't exist yet, create it before writing to it.

## Navigation

- Start here when you need to understand what's available
- Follow links within files rather than listing the entire tree
- Use `search_files` to locate specific content by keyword if you don't know the path

# Research

Research materials are saved in `playground/research/` organized by date.

## Directory convention

```
playground/research/
  YYYY-MM-DD/
    <topic-slug>.md    ← one file per source or topic
    summary.md         ← optional: synthesis of all files in this session
```

Example: `playground/research/2026-04-05/observational-memory.md`

Use today's date (available in `<context>` in your current message) for the directory name.

## File format

Each research file should contain:

1. **Source** — URL, title, or origin of the material
2. **Key points** — concise bullets, not copy-paste
3. **Relevance** — one sentence on why this matters for the current task

Keep files short. If a topic has many sources, write one file per source and one `summary.md` that synthesizes them.

## Referencing across sessions

To hand off research to a future session or another chat:
- Write a `summary.md` with a list of files and one-line descriptions
- The future session reads `summary.md` first, then drills into individual files as needed
